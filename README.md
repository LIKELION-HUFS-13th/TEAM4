# ✉️ Later To Me: 미래의 나에게 보내는 한 통의 편지

**Later To Me**는 '나중에'를 뜻하는 Later와 편지를 뜻하는 'Letter'의 발음이 비슷한 것에서 착안한 것으로,

n일 뒤의 나에게 편지를 쓰고, n일 뒤 그 편지를 열어볼 수 있는 서비스입니다.

## 📃 서비스 소개

- "기억을 남기세요. 마음을 꺼내보세요."
  - 지금의 나는, 미래의 나에게 어떤 말을 전하고 싶을까요?
  - “n년 뒤 나에게 쓰는 편지”는 당신이 스스로에게 보내는 편지를 기록하고, 잊혀질 뻔한 감정과 생각들을 다시 꺼내볼 수 있게 도와주는 타임캡슐입니다.

- “무엇을 써야 할지 몰라도 괜찮아요.”
  - 오늘 느낀 작은 기쁨, 문득 떠오른 고민, 아무 말 대잔치라도 좋아요.
  - 당신의 말 한 줄 한 줄이 미래의 나에게 큰 힘이 되어줄 테니까요.

- “바쁜 하루 속에서도 잠깐 멈춰, 스스로에게 말을 걸어보세요.”
  - n년 뒤, 당신의 편지가 조용히 도착할 그날까지 우리는 소중한 마음을 잘 간직하고 있을게요.

- 그때의 나와 지금의 나를 이어주는 한 통의 편지. “n년 뒤 나에게 쓰는 편지”, 당신의 시간을 기록하세요.
